# 10. Развёртывание системы (Deployment)

## 10.1 Общее описание

В данном разделе описан процесс развёртывания **backend-системы RoadVision AI**.

Система построена на **Django REST Framework**, включает AI-модуль компьютерного зрения, обработку очередей задач через Celery и работу с RTSP-потоками в реальном времени.

## 10.3 Установка проекта

### Пошаговая инструкция

1. **Клонирование репозитория**
   ```bash
   git clone <repository_url>
   cd roadvision-ai
   ```
2. **Создание виртуального окружения**
   ```bash
   python -m venv venv
   ```
3. **Активация окружения Bash**
   ```bash
   # Linux / macOS
   source venv/bin/activate

   # Windows
   venv\Scripts\activate
   ```
4. **Установка зависимостей**
   ```bash
   pip install -r requirements.txt
   ```
5. **Применение миграций**
   ```bash
   python manage.py migrate
   ```
6. **Создание суперпользователя**
   ```bash
   python manage.py createsuperuser
   ```
7. **Запуск сервера разработки**
   ```bash
   python manage.py runserver
   ```
## 10.4 Запуск Redis
### Redis необходим для работы Celery (фоновые задачи).
   ```bash
     redis-server
   ```
## 10.5 Запуск Celery Worker
   ```bash
     celery -A config worker -l info
   ```
## 10.6 Работа со стримингом (RTSP)
   ```bash
     celery -A config worker -l info
   ```
### 1. Добавьте камеру через API
### 2. Запустите поток через эндпоинт:
   ```bash
     POST /api/streams/start/
   ```
### RTSP-потоки обрабатываются отдельным сервисом внутри backend.
